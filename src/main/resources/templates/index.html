Quem somos ? Oque fazemos ? Como fazemos ? Quem desenvolveu ?
Olá! Somos estudantes do 4° semestre do curso Análise e Desenvolvimento de Sistemas, voltado a desenvolver um E-Commerce de produtos eletrônicos.
Realizamos uma simulação de um E-commerce, onde temos três tipos de usuários como usuário cliente o qual irá efetuar suas compras pelo sistema, o usuário estoquista que irá realizar toda a gestão de produtos no sistema e o usuário administrador, que irá realizar toda gestão de usuários quanto a gestão de produtos dentro do Back Office. Tendo assim cada usuário suas devidas permissões, e também permitindo que qualquer usuário anônimo da web possa visualizar os produtos e se cadastrar como usuário cliente no sistema.

Utilizamos as seguintes ferramentas de desenvolvimento:

  • Java 8
  • Spring Framework
  • Thymeleaf
  • MySQL 8
  • HTML5, CSS3 e JS
  • Bootstrap 4.5
  • Azure DevOps
